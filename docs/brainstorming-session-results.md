## Design System (Analogical Thinking Results)

### Inspirations
| Source | What to Take |
|--------|--------------|
| **Linear.app** | Sharp typography, smooth animations, precision feel, beautiful hover states, generous spacing, premium quality |
| **shadcn/ui** | Component style, light theme execution, subtle shadows, rounded corners |
| **WhatsApp/Google/Swiggy** | Clean, modern, trustworthy feel |

### Design Specifications

| Element | Specification |
|---------|---------------|
| **Theme** | Light (white / #fafafa background) |
| **Primary Color** | Navy blue (#1a2744) from logo |
| **Accent Color** | Metallic blue gradient from logo |
| **Typography** | Inter or Geist — clean, sharp, modern |
| **Components** | shadcn/ui style — cards, subtle borders, smooth hover states |
| **Animations** | Smooth fade-ins, subtle hover lifts, purposeful (not flashy) |
| **Spacing** | Generous white space (premium feel) |
| **Feel** | Premium, precise, modern, fast |

### Design Principles
- NO dark theme — light, bright, premium
- Linear.app quality execution
- Every pixel builds trust
- Design IS the sales pitch

## Conversion Strategy (Role Playing Results)

### Primary Insight
> Premium Design = Trust = Conversion

The website itself is the sales pitch. Design quality proves work quality.

### Visitor Psychology
| Visitor Thinks | Because They See |
|----------------|------------------|
| "They're professional" | Clean, polished design |
| "They're precise" | Pixel-perfect details |
| "They're modern" | Updated tech, smooth animations |
| "I can trust them" | Premium feel = premium work |
| "I want to work with them" | → Contacts Molvexa |

### Visitor Journey
```
LAND ON SITE
     ↓
"Wow, this looks premium"
     ↓
"If their website is this good..."
     ↓
"...their mold work must be excellent"
     ↓
TRUST ESTABLISHED
     ↓
Browse Services → See Expertise
     ↓
CONTACT (form / call / WhatsApp)
```

### Contact Methods (All)
| Method | Placement |
|--------|-----------|
| Contact Form | Contact page + CTA sections |
| Phone | Header + Contact page + Footer |
| WhatsApp | Floating button (always visible) |
| Email | Contact page + Footer |
